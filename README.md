#  EventQueue – Mini Event Management Website (Frontend)

EventQueue is a **static frontend event management website** inspired by platforms like Eventbrite. The project focuses on building a clean and user-friendly interface where users can browse events, view event details, and simulate event registration.

This project is implemented using only **HTML, CSS, and JavaScript**, with no backend connectivity or database integration.

---

##  Project Overview

EventQueue demonstrates how a real-world event listing and registration platform can be visually and functionally represented using frontend technologies alone. The website allows users to explore events and navigate between different pages smoothly while keeping the scope simple and achievable.

The project emphasizes:
- UI/UX design
- Frontend logic
- Basic interactivity using JavaScript

---

##  Features

### 1. Event Listing Page
- Displays multiple upcoming events in a list or grid layout
- Each event card includes:
  - Event title
  - Date and venue
  - Short description or poster image

### 2. Event Detail Page
- Clicking on an event opens a detailed view
- Displays:
  - Event description
  - Time and location
  - Event-related information

### 3. RSVP / Register Button (Frontend Simulation)
- Users can click a **Join / Register** button
- Registration is simulated using JavaScript (no backend or data storage)

### 4. Navigation
- Simple and intuitive navigation between:
  - Home / event listing page
  - Event detail pages
  - Find events
  - Create events
  - Help

---

##  Tech Stack

- **HTML5** – Structure and content
- **CSS3** – Styling, layout, and responsiveness
- **JavaScript (Vanilla JS)** – Page interactivity and event handling

>  Note: This is a **static website**.  
> - No backend server  
> - No database  
> - No real-time attendee updates
